Jaunt
=====

> Quickly travel object tree structures


Summary
-------


Installation
------------

~~~
% npm install jaunt
~~~


### Dependencies ###

These are installed when **jaunt** is installed.

~~~
protean: 0.1.x
~~~


### Development Dependencies ###

Installed when you run `npm link` in the package directory.

~~~
mocha:  1.7.x
should: 1.2.x
sake:   0.1.x
ejs:    0.8.x
~~~
